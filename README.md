# Warsztat docker

## Wymagania

Przed przystąpieniem do warsztatu nalezy:

1. zainstalowac klienta git
2. zainstalowac Docker:
  * Dla Linux: https://get.docker.com/
  * Dla Windows: https://docs.docker.com/docker-for-windows/install/
  * Dla MacOS: https://docs.docker.com/docker-for-mac/install/

3. zainstalowac docker-compose: https://docs.docker.com/compose/install/
4. przejśc conajmniej szkolenie https://training.play-with-docker.com/beginner-linux/
